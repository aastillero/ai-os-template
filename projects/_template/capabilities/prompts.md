# Project Prompts

Reusable prompts for this project.

## Daily project focus

```text
Using this project workspace, review context/current-state.md, context/status.md, context/raid-log.md, connections/stakeholders.md, and context/work-plan.md. Give me the top 3 project focus items today, why each matters, and the next action for each.
```

## Status update draft

```text
Using this project workspace, draft a concise status update. Ground it in current status, work completed, blockers, risks, decisions needed, and next steps. Do not invent dates or commitments.
```

## Risk review

```text
Using context/raid-log.md and connections/dependencies.md, identify the highest-risk open items, what evidence supports them, and the next action needed.
```
