# Milestone Checkpoint

Use this before or after a milestone, release, handoff, or major review.

## Prompt

```text
Using this project workspace, prepare a milestone checkpoint. Review scope, definition of done, current status, open risks, dependencies, decisions, stakeholder readiness, and evidence. Tell me whether the milestone looks ready, what is missing, and what should happen next.
```

## Readiness checklist

- [ ] Scope is clear
- [ ] Exit criteria are defined
- [ ] Evidence exists for completed work
- [ ] Open risks are understood
- [ ] Dependencies are resolved or accepted
- [ ] Stakeholder communication is ready
- [ ] Follow-up ownership is clear
