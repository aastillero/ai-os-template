# Daily Project Focus

Use this when starting focused work on this project.

## Prompt

```text
Using this project workspace, help me plan today's project focus. Review the project brief, current state, status, RAID log, stakeholders, dependencies, and work plan. Give me the top 3 items that need attention today, why each matters, and the next action I should take.
```

## Checklist

- [ ] Current state is still accurate
- [ ] Status is current
- [ ] Blockers and risks are visible
- [ ] Decisions needed are clear
- [ ] Stakeholder follow-ups are clear
- [ ] Today's next actions are realistic
