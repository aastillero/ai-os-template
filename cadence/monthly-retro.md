# Monthly Retro

Use this monthly or at the end of a work cycle to improve how you operate in the active role.

## Suggested prompt

```text
Using this workspace and the active role profile, help me review the last month or work cycle.

Review the 4C pattern across active projects:
1. Context: what was stale, missing, or especially useful?
2. Connections: which stakeholder, dependency, system, or communication patterns mattered most?
3. Capabilities: which repeated workflows or prompts should be improved or considered for future skills?
4. Cadence: which daily, weekly, or milestone rituals helped or wasted time?

Then identify:
1. recurring risks
2. repeated blockers
3. decisions that took too long
4. communication patterns that worked
5. communication patterns that caused confusion
6. one workflow I should improve next month
7. one task that may deserve a future Copilot skill
```

## Output to capture

Add durable lessons to the relevant project/workstream files, templates, role profile, or project `capabilities/skill-backlog.md`.
